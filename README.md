# AI Master Class 1010

This repository contains the materials for the AI Master Class 1010. It provides a structured collection of Jupyter notebooks designed to introduce fundamental and advanced concepts in Artificial Intelligence, Machine Learning, and Deep Learning.

## Overview

The goal of this repository is to support students with practical, hands-on examples. Each notebook focuses on a specific concept and includes explanations, code implementations, and exercises.

## Repository Structure

The content is organised in a progressive manner:

* `01-introduction/` – Overview of Artificial Intelligence and course setup
* `02-python-basics/` – Python fundamentals for data science
* `03-numpy-pandas/` – Data manipulation and analysis
* `04-ml-basics/` – Core machine learning concepts
* `05-deep-learning/` – Introduction to neural networks

Additional resources such as datasets may be included in a separate `datasets/` directory.

## Requirements

To run the notebooks locally, install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ai-masterclass-1010.git
cd ai-masterclass-1010
```

2. Launch Jupyter Notebook:

```bash
jupyter notebook
```

3. Open the notebooks and follow them in order.

## Guidelines

* Follow the notebooks sequentially for best understanding
* Read the explanations carefully before running the code
* Modify and experiment with the code to reinforce learning
